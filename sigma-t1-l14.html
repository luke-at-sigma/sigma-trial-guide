Add a KPI and bar chart

Go beyond tables — add a single-number KPI tile and a grouped bar chart to your workbook canvas.

KPI and bar chart (5 min) KPI element · Bar chart · Chart formatting
Try with Sigma Assistant — Add a KPI showing total revenue and a bar chart of revenue by brand
Concept
Visualization elements on the canvas

Sigma workbooks are made up of elements — tables, charts, controls, text, images, and more. Every element is data-connected: it draws from a table in your workbook (which in turn queries the warehouse). Change the underlying data or apply a filter control, and every visualization updates live.

Two of the most common elements are:

  • KPI — a single large number with an optional comparison value and trend indicator. Great for totals, counts, or percentages at a glance.
  • Bar chart — a grouped or stacked chart for comparing values across categories. Built directly from columns in a table without needing to pre-aggregate in SQL.
Step 1
Add a KPI element

In Edit mode, click Add element → Visualizations → KPI. Sigma places an empty KPI tile on the canvas. In the chart editor panel on the right:

  • Set Source to the table that has your metric column (e.g., the Orders table)
  • Drag your metric column — e.g., Revenue — into the Value field. Sigma automatically aggregates it as Sum([Revenue])
  • Optionally drag a date column into Comparison period to show period-over-period change
Add a Total Revenue KPI
Step 2
Format the KPI

With the KPI element selected, open the Format tab in the editor panel. Key formatting options:

  • Number format — choose Currency, Compact (1.2M), Percentage, or custom formats like $#,##0
  • Label — add a title below the number, e.g., "Total Revenue"
  • Comparison label — customize the "vs. prior period" text
  • Color rules — make the comparison value green when positive, red when negative
KPI tile showing $4.2M with a green +12% vs. prior month comparison
Step 3
Add a bar chart

Click Add element → Visualizations → Bar chart. In the chart editor:

  • Set Source to your Orders table
  • Drag Region (or any category column) into the X-axis field
  • Drag Revenue into the Y-axis field — Sigma aggregates it as Sum automatically
  • Optionally drag Product Category into Color to create a grouped bar chart

Sigma computes the aggregation in SQL — no manual GROUP BY needed.

Add a Revenue by Region bar chart
Bar chart: East, West, Central bars with Revenue on the Y-axis
Step 4
Chart formatting options

Open the Format tab in the chart editor to customize the bar chart:

  • Chart orientation — switch to horizontal bars when category labels are long
  • Axis labels and title — add axis titles and customize tick formatting
  • Data labels — show value labels on top of each bar
  • Color palette — choose from Sigma's built-in palettes or set custom hex colors
  • Sort — sort bars by value (descending) to create a rank chart
Charts update live. Because all chart data is computed directly in the warehouse, every chart responds instantly when a control filter changes or when new data lands in your warehouse tables. No refresh or manual query needed.
