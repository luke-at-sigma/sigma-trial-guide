Learning Paths by Role

Not sure where to start — or rolling Sigma out to your team? Here's what each role should focus on, with recommended tracks and key skills.

Administrator
Configures Sigma for the organization — manages connections, users, permissions, and AI features.

Recommended track
Admin Setup
Track 3 · 8 lessons
Key skills you'll develop
Connect Sigma to your cloud data warehouse
Manage user accounts, licenses, and team access
Configure AI features including Sigma Assistant and Sigma Agents
Set up governance — row-level security, usage reporting, and audit logging
Customize the Sigma environment for your organization's needs

Builder
Creates dashboards, reports, and analyses — either extending existing workbooks or building from a live data connection.
Build license
Recommended track sequence
Working in Sigma
Track 1 · 16 lessons
Build from Scratch
Track 2 · 13 lessons
Key skills you'll develop
Navigate workbook modes (View, Explore, Edit) and understand how they affect data
Write formulas, apply filters, group and aggregate data
Build charts, KPI tiles, and combo visualizations with formatting
Connect to data sources, write custom SQL, and join tables
Use Sigma Assistant and Sigma Agents to build analyses with natural language
Create linked Input Tables for data writeback and annotation workflows

Explorer
Works with existing content — explores data, builds custom views, and applies formulas without starting from scratch.
Analyze license (or Build)
Recommended track
Working in Sigma
Track 1 · 16 lessons
Key skills you'll develop
Understand View, Explore, and Edit modes and when to use each
Use Explore mode to dig into published content without affecting the original
Write custom formulas and add calculated columns to existing tables
Customize chart styles, colors, and axis settings
Save and share custom views so your team can benefit from your analysis
